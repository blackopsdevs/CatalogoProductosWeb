# Catalogo de productos Web {Product catalog web}

_Este mini-proyecto pretende exponer el catalogo de productos [Catalogo de productos](https://github.com/blackopsdevs/CatalogoProductos) en un menu de productos de al sigueinte forma:_

* **Menu Departamentos**
* **Menu Categorias**
* **Menu Productos**
